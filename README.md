### Requirements
- Burpsuite
- Java

### Install Plugin

	Download Jar https://github.com/d3vilbug/HackBar/releases and add in burpsuite

### Tested on
- Burpsuite 1.7.36
- Windows 7/8.1/10
- Kali linux (2013.3)
